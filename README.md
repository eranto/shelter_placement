# Road Shelter Placement — Israel

Optimal placement of roadside bomb-shelters (מיגוניות) on Israeli roads using the **Gonzalez Farthest-First (k-center)** algorithm, with deployment prioritisation based on missile alert density and proximity to the Lebanon and Gaza borders.

## Live Map

Open [shelters_priority_final.html](shelters_priority_final.html) locally to explore the final shelter placement.

## What it does

1. **Fetches** motorway, trunk, primary, and secondary road geometry from OpenStreetMap (Overpass API)
2. **Filters** to 1967 Green Line Israel only — excludes West Bank, Gaza Strip, Golan Heights, Sinai, and Lebanon (polygon + line border tests via `israel_1967_filter.py`)
3. **Builds** a road-network graph with junction cross-edges and travel-time weights
4. **Places shelters** using Gonzalez k-center so every road point is within the time limit:
   - Standard roads: **5-minute** access
   - Jerusalem corridor: **3-minute** access
   - Junction/interchange nodes get a placement preference bonus
5. **Post-deduplication** removes shelters within 2.5 km of each other
6. **Urban relocation** moves shelters near residential areas (≥ 400 missile alerts) to rural road nodes on the same route
7. **Composite ranking** scores each shelter for deployment priority:
   - **50%** Mar–Apr 2026 missile alert density within 15 km
   - **50%** proximity to the Lebanon or Gaza border (linear decay over 50 km)

Northern border roads near Lebanon rank highest due to border proximity combined with high alert density. Central roads (Route 1, 44, 46) rank high on alert density. Gaza-envelope roads rank high on border proximity.

## Setup

```bash
pip install -r requirements.txt
```

### Run the full pipeline

```bash
# Step 1 — fetch OSM road data and run Gonzalez placement (~5 min, downloads on first run)
python run_priority_placement.py

# Step 2 — dedup, relocate, score, and export map + CSV (~2 min)
python shelter_priority_pipeline.py
```

OSM road data is cached in `osm_roads_traffic_cache.pkl` after the first run.  
Alert data requires `/tmp/israel_alerts.csv` from [github.com/dleshem/israel-alerts-data](https://github.com/dleshem/israel-alerts-data) and `/tmp/geocoded_cities.json` (generated by the geocoding step in `road_shelters_traffic.py`).

## Algorithm reference

> Gonzalez, T.F. (1985). Clustering to minimize the maximum intercluster distance. *Theoretical Computer Science*, 38, 293–306.

Greedy 2-approximation for the k-center problem: place each new shelter at the node currently farthest from any existing shelter, weighted by zone time limit and road capacity. Runs in O(k · E log V) via repeated Dijkstra.
